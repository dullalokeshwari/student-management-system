# 🎓 Student Management System (Python + SQLite)

## 📌 Project Description
This is a simple Student Management System built using Python and SQLite database.  
It performs basic CRUD operations like Add, View, Update, and Delete student records.

---

## 🚀 Features
- Add new student details
- View all student records
- Update student information
- Delete student records
- Stores data using SQLite database

---

## 🛠️ Technologies Used
- Python
- SQLite (built-in database)
- VS Code
